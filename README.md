## 🚀 Main Project

### 📌 채용공고 맞춤 추천 서비스

> 사용자 이력서를 기반으로 적합한 채용공고를 추천하는 데이터 기반 웹 서비스

#### 🔧 주요 기능

* 채용공고 크롤링 (데이터 수집 자동화)
* 키워드 / 필터 기반 검색 기능
* 사용자 이력서 입력 기능
* 공고-이력서 간 유사도 계산 기반 추천

#### ⚙️ 기술 스택

* Backend: Python, Django
* Data: 크롤링, 텍스트 유사도 분석
* DB: SQLite / (추후 확장 가능)
* 기타: REST API 설계

#### 💡 핵심 구현 포인트

* 크롤링 데이터를 정제하여 추천 시스템에 활용
* 텍스트 유사도 기반 매칭 로직 직접 구현
* 단순 기능 구현이 아닌 “서비스 흐름” 중심 설계
